- violate shared contracts
- bypass API gateway, RBAC, analytics hooks, or monitoring hooks

**Always:**

- reuse shared modules
- document APIs, version contracts
- generate migrations, tests, telemetry, observability hooks
- optimize for mobile bandwidth and low RAM Android devices

Platform folders are generated by Flutter CLI on first setup.

Run from repo root:

```powershell
powershell -ExecutionPolicy Bypass -File scripts/setup-flutter-mobile.ps1
```

Or:

```powershell
cd apps/mobile
flutter create . --org com.streamheaven --project-name stream_heaven_mobile
```

This creates standard `android/` and `ios/` project files required to build the app.

# Stream Heaven — Shared Packages

Cross-app libraries. No feature-specific business logic — only contracts, types, utils, and design system.

| Package | Purpose | Phase 1 Priority |
|---------|---------|------------------|
| shared-contracts | OpenAPI, AsyncAPI, JSON Schema | **P0** |
| shared-types | Domain enums, error codes | **P0** |
| shared-utils | Retry, locale, idempotency helpers | **P1** |
| design-system | Flutter tokens, components, themes | **P1** |

Contract-first: read [`docs/shared-contracts-overview.md`](../docs/shared-contracts-overview.md) before any API implementation.
